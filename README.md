# Regex Lab

Interactive regular expression playground built with **Vue 3**, **Pinia**, and **VueUse**.

## Features

- Live match highlighting on sample text
- Capture group inspection (numbered and named)
- Pattern presets (email, URL, phone, dates, and more)
- Heuristic token explanations
- Flag toggles (global, ignore case, multiline, dotall)

## Run

```bash
npm install
npm run dev
```

## Stack

Vue 3 · Pinia · VueUse · Tailwind CSS v4 · Vite
